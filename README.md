# Blueline Tour Package System – Defect Reporting (Jira)

## 📌 Project Description
This project focuses on identifying and reporting defects in a tour booking application.

## 🐞 Work Done
- Identified defects in booking and payment modules  
- Created detailed bug reports with clear steps to reproduce  
- Documented Expected vs Actual results  
- Assigned severity levels based on defect impact  
- Tracked and managed defects using Jira  

## 🛠 Tools Used
- Jira (Defect Tracking)  
- Manual Testing  

## 📂 Repository Contents
- Bug_Reports/  

## 🎯 Objective
To demonstrate practical knowledge of defect reporting, bug tracking, and structured documentation in a QA process.
